# Community Bounty Board

A decentralized task and project bounty system for community-driven development on the Stacks blockchain.

## Features

- Sponsors can post bounties with reward amounts and requirements
- Contributors can claim bounties and submit work
- Head moderator can approve bounty completions
- Complete contribution tracking and reward distribution

## Smart Contract Functions

### Public Functions
- `post-bounty`: Create new bounty listing with rewards
- `claim-bounty`: Accept bounty task as contributor
- `complete-bounty`: Approve bounty completion (moderator only)

### Read-Only Functions
- `get-bounty`: Retrieve bounty listing details
- `get-contribution-log`: Access contribution records
- `get-contribution-count`: Get total bounty contributions
